# CrackHash

## 📌 Описание проекта

**CrackHash** --- распределённая система для подбора строк по их
MD5-хэшу методом полного перебора (brute-force).

Система построена по архитектуре **master-worker** и состоит из двух
компонентов:

-   **Manager** --- принимает запросы, разбивает задачу на части,
    отправляет воркерам и агрегирует результаты\
-   **Worker** --- перебирает строки и вычисляет их MD5

------------------------------------------------------------------------

## Как работает система

1.  Клиент отправляет MD5-хэш и максимальную длину строки в Manager\
2.  Manager создаёт уникальный `requestId`\
3.  Пространство поиска делится на части по количеству воркеров\
4.  Каждому воркеру отправляется своя часть\
5.  Воркеры перебирают комбинации и ищут совпадения\
6.  Результаты отправляются обратно в Manager\
7.  Клиент получает статус задачи по `requestId`

------------------------------------------------------------------------

## Основные возможности

-   Распределённые вычисления\
-   Масштабирование за счёт увеличения количества воркеров\
-   Асинхронная обработка задач\
-   Отслеживание статуса по `requestId`\
-   Кэширование результатов\
-   Обработка отказов (часть воркеров может упасть)

------------------------------------------------------------------------

##  Ограничения

-   Используется только алгоритм **MD5**\
-   Алфавит по умолчанию: a-z + 0-9

## Client API

### Создание задачи

POST /api/hash/crack

``` json
{
  "hash": "5d41402abc4b2a76b9719d911017c592",
  "maxLength": 5
}
```

Ответ:

``` json
{
  "requestId": "uuid"
}
```

------------------------------------------------------------------------

### Проверка статуса

GET /api/hash/status?requestId=`<uuid>`{=html}

Статусы:

IN_PROGRESS --- выполняется\
READY --- завершено\
PARTIAL_RESULT --- частичный результат\
ERROR --- ошибка

------------------------------------------------------------------------

## Internal API

### Manager → Worker

POST /internal/api/worker/hash/crack/task

(XML)

### Worker → Manager

PATCH /internal/api/manager/hash/crack/request

(XML)
